# Change Log

All notable changes to the **Ainara** extension will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/), and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

---

## [1.0.2] - 2026-05-30

### Added
- **License:** Added MIT License to the project.

### Changed
- **Asset Refactoring:** Updated icons and references from `logo.png` to `icon.png`.
- **Showcase Structure:** Redesigned the showcase layout to use specific screenshot files (`screenshots/`) for each format.

## [1.0.0] - 2026-05-29

### Added
- **Initial Release** of the **Ainara Dark** theme.
- **Color System:** Designed around deep petroleum green backgrounds, slate secondary panels, and clean teal highlights.
- **Syntax Highlighting:** High-contrast rules tailored for TypeScript, JavaScript, Python, HTML, CSS, JSON, and Markdown.
- **Marketplace Integration:** Configured all metadata, keywords, and `.vscodeignore` exclusions required for a clean publishing bundle.
- **Developer Assets:** Created sample files in the `tests/` directory to facilitate visual testing and screenshot capture.
- **Documentation:** Added a comprehensive English `README.md` with guidelines on recommended settings, styling overrides, and showcase placeholders.
